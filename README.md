# 🎌 Anime Analysis — MyAnimeList

## Overview
Data analysis project on animes and mangas using the MyAnimeList 
dataset (14,000+ entries), covering the full data analyst workflow 
from raw data to interactive dashboard.

## Tech Stack
- **Python / JupyterLab** — Data cleaning and exploratory analysis
- **Google BigQuery** — Cloud storage and SQL querying
- **Power BI** — Interactive visualization dashboard

## Key Insights
- Score distribution follows a bell curve centered around 6.5/10
- Comedy is the most produced genre but not the highest rated
- Studios with the highest output are not necessarily the best rated
- Positive correlation between popularity and score
- Scores peaked around 2012 then declined (selection bias)

## Project Structure
- `analyse_manga.ipynb` — Full Jupyter Notebook with code and charts
- `anime_clean.csv` — Cleaned dataset
- `dashboard_screenshot.png` — Power BI dashboard
